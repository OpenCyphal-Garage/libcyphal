Detailed Description

Optional utilites to provide introspection into libuavcan objects and to provide metrics for library performance. Facilities defined in this header can be compiled out of production code and should be used for debugging or targeted testing activities.

Macro Definition Documentation

◆ LIBUAVCAN_INTROSPECTION_ENABLE_ASSERT

#define LIBUAVCAN_INTROSPECTION_ENABLE_ASSERT   0

Enable runtime asserts within libuavcan.

Obviously, this is not a reccommended thing to turn on in production code but these asserts are used for some library tests and may help diagnose bugs if enabled in a special build of your software.
